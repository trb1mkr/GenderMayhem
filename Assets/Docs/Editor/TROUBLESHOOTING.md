# 🔧 Troubleshooting

Документ содержит известные проблемы и их решения при разработке проекта.

## 🐛 Частые проблемы

- Когда возникает ошибка про zero size, то надо удалить папку TextMesh Pro и импортировать заново
- Включённая виртуализация в BIOS не даёт сделать Attach дебаггера к Unity
- Camera.main не возвращает объекты, которым после старта сцены присвоен тег MainCamera
- Если в анимации не поставить начальный фрейм (0), то анимация перескочит в конец
- Достаточно иметь 1 анимацию, где анимируется переменная, чтобы переменную нельзя было изменять через скрипт 
- Avoid коллайдер запрещает бить и стрелять, но при этом, если удар уже в процессе, то он не останавливается
- Если проблемы с производительностью из-за физики, то смотри в Time - Fixed timestep
