# 🎮 Правила работы в Unity

Правила работы в редакторе Unity.

## 🏷️ Иконки и Gizmo

- Использовать иконки GameObject в редакторе, где можно, вместо Gizmo. Если объект можно пометить иконкой (например, камера, спавнер, точка интереса) — использовать встроенные иконки Unity (через Inspector → GameObject Icon), а не рисовать кастомный `OnDrawGizmos`.
- Gizmo использовать только когда иконка недостаточна (нужно показать зону, радиус, направление, связь между объектами).

## 📝 Логирование

- Использовать `CustomLogger.Log` / `LogWarning` / `LogError` из `Azen.Logger` вместо стандартного `Debug.Log` / `Debug.LogWarning` / `Debug.LogError`.
- Выбирать подходящую `LogCategory` для каждого сообщения:
- Не использовать `Debug.Log` в production-коде. Допускается только в auto-generated файлах (например, `InputActions.cs`).
